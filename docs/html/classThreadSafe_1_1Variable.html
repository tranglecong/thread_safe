Detailed Description

template<typename T>
class ThreadSafe::Variable< T >

A thread-safe wrapper for a variable of type T.

This class allows safe concurrent access to a variable by using a mutex to guard its access. It provides thread-safe assignment, comparison, and access to the encapsulated variable.

Template Parameters
TThe type of the variable to be protected.
